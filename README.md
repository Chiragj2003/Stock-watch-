# Stock Watch

This is a repository for a FullStack stock market prices watching app.

Live app: https://stock-watch-phi.vercel.app

## About

Stock Watch is a web application for tracking stock market prices in real time, browsing market highlights, and exploring stock recommendations through a modern full-stack workflow.

## Tech Stack

- Next.js
- TypeScript
- Tailwind CSS

## Features

- Real-time stock price watching interface
- Market overview section
- Stock recommendations page
- Search-first stock browsing experience

## Getting Started

1. Install dependencies:

```bash
npm install
```

2. Start the development server:

```bash
npm run dev
```

3. Open http://localhost:3000 in your browser.

## Production

The app is deployed on Vercel at:

https://stock-watch-phi.vercel.app
